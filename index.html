What is Hypatia?

Hypatia is a FREE open-source modern Learning Management System based on the powerful an popular Facebook's React framework and Node.js. Hypatia aims at solving the issues that current LMS have, like having bad UX, very old and cluttered design or not working properly on mobile devices.

Who can use it?

Hypatia is open-source software and can be used by anyone willing to start an educational project online. It's philosophy is to help people to bring education for everyone in the world. For that reason, it's completely FREE, without any hidden cost or license. It is designed with universities and academies in mind, which means you will find features that are usually required for these kind of projects. There's no size limit, Hypatia is flexible and scalable. It can be used for small academies (5 - 100 students) and for big universities (1K - 100K).

Creating pages manually
